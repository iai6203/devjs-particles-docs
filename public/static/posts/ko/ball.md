# Ball Config

다음은 `Ball` 효과를 커스터마이징 할 수 있는 설정값입니다.

~~~javascript
{
  // `count`는 공의 개수를 설정합니다.
  count: 200 // 기본값
  
  // `distance`는 공이 커지는 범위를 설정합니다.
  distance: 200 // 기본값
  
  // `radius`는 공의 지름을 설정합니다.
  radius: 20 // 기본값
  
  // `maxRadius`는 공이 확장될 최대 지름을 설정합니다.
  maxRadius: 40 // 기본값
  
  // `colors`는 공의 색상을 설정합니다.
  colors: ['#000'] // 기본값
}
~~~
